# Requirements

正式な要件原文は `docs/elevator-game-requirements.md` を参照します。

今回の実装対象は以下です。

- 単一ルームの参加受付、UUID識別、名前重複チェック。
- ホスト認証、ステージ進行、締切カウントダウン、集計、ランキング。
- プレイヤーの乗車階・降車階・予想イベント回答・棄権。
- エレベータールールに基づくスコア計算。
- E1〜E6イベントの初版対応。
- Skill値のステージ別計算と現在Skill値計算。
- スクリーン表示と結果発表アニメーションの初版。
- GAS Web App APIとSpreadsheet初期化。
- 実装判断、逸脱、未解決点の `docs/implementation-notes.html` 記録。

未実装または暫定実装の詳細は `docs/implementation-notes.html` に記録しています。
